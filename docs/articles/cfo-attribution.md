## Summary

Stop reporting impressions to the C-suite. Start showing how earned media moves pipeline. Most marketing ROI reports fail because they answer questions executives aren't asking. Here's the exact framework that builds CFO trust: pattern-based insights over perfect attribution.

## Key Insights

- Executives mistrust attribution because most models are unintelligible
- Pattern-based insights ("Deals with earned media closed 27% faster") build more trust than complex multi-touch models
- CFOs want revenue-mapped outcomes: pipeline contribution, win rates, deal velocity
- Directional efficiency trends demonstrate control and scalability
- Add "Earned Media Exposure" field to CRM today—in 90 days you'll have pattern data
