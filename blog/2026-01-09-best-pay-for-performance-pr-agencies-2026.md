## Pay-for-Performance PR vs Performance PR

While these terms are often used interchangeably, there's an important distinction: pay-for-performance PR refers to the payment model, while [performance PR](https://blog.authoritytech.io/performance-pr-future-media-relations-2026) refers to the results focus.

### Key Differentiation

- **Pay-for-Performance PR:** Payment model where agencies only get paid for placements/results

- **Performance PR:** Results focus where agencies track performance metrics and KPIs

Both are better than retainers, but they emphasize different aspects. Pay-for-performance PR emphasizes the payment structure (how you pay), while performance PR emphasizes the metrics focus (what you measure).

Many agencies claim to offer "performance PR" but still use retainer pricing. True pay-for-performance PR agencies charge per placement, not monthly fees. This distinction matters because it determines where the risk lies and how incentives are aligned.

### When to Use Each

Choose pay-for-performance PR when you want payment tied directly to outcomes. This model works best when you need guaranteed placements and transparent pricing. Choose performance PR when you want to track performance metrics and KPIs, even if you're still paying monthly retainers.
